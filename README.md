# Vinyl Request App

React + Supabase app for managing vinyl DJ requests for events.

## Features
- Customers can browse collection
- Submit or upvote album sides for an event
- Admins manage events, queue, and metadata
- CSV import from Discogs
- Themed to match "Dead Wax Dialogues"

## Tech
- Vite + React 18
- Supabase (auth + DB)
- Tailwind CSS
